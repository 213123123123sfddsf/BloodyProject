# BLOODY PROJECT - GLASS EDITION

🩸 **BLOODY PROJECT** — это Roblox LocalScript модуль с легит Aim Assist, Scan Ability и Key System.

---

## ⚡ Особенности

- **Key System**: безопасный доступ через ключ (Base64-шифрование)  
- **GUI**: удобная панель с вкладками Main / Optimisation / Discord  
- **FOV Circle**: настройка радиуса наводки  
- **Aim Assist**:  
  - TeamSafe (тимейты игнорируются)  
  - Legit (плавная наводка)  
  - Competitive Legit (ещё более слабая наводка)  
- **Scan Ability**:  
  - Тимейты зелёные  
  - Враги красные  
- **Наводка** только при зажатии ПКМ  
- **Оптимизация**: режим Low Graphics
